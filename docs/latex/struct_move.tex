\doxysubsection{Detailed Description}
Represents a possible move for a \mbox{\hyperlink{struct_tower}{Tower}}. 

A user can perform a move, depending on the {\ttfamily conquer} variable, either only moves the tower in the origin cell to the destination, or also conquers the tower in the eaten cell.
