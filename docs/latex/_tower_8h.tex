\doxysubsubsection{\texorpdfstring{Tower}{Tower}}
{\footnotesize\ttfamily typedef struct \mbox{\hyperlink{struct_tower}{Tower}} \mbox{\hyperlink{struct_tower}{Tower}}}



Array of \mbox{\hyperlink{struct_piece}{Piece}}. 

Every piece used in the game is a tower, starting from height = 1, it can grow taller by conquering different pieces. The first element of the tower, placed in position 0, is the one that controls the tower, also indicating it\textquotesingle{}s color. When the tower gets conquered, that is the piece that gets conquered along with its same color pieces, while the rest of the pieces (of the opposite color) are left in position to create a new tower. \mbox{\Hypertarget{_tower_8h_a460b8c527f24bb21e504b2761dc13278}\label{_tower_8h_a460b8c527f24bb21e504b2761dc13278}}
