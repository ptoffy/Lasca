\doxysubsubsection{\texorpdfstring{Board}{Board}}
{\footnotesize\ttfamily typedef struct \mbox{\hyperlink{struct_board}{Board}} \mbox{\hyperlink{struct_board}{Board}}}



Game field. 

\mbox{\hyperlink{struct_board}{Board}} is the game field that contains every \mbox{\hyperlink{struct_cell}{Cell}} and every \mbox{\hyperlink{struct_tower}{Tower}} that make up the game. The cells pointer is an array that contains all the \mbox{\hyperlink{struct_cell}{Cell}} of the board, with the size of rows $\ast$ columns, in classic Lasca it is 7 $\ast$ 7, meaning there are 49 cells. Lasca is played only on white cells, in fact the actually used cells are just 25.
