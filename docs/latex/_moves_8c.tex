\doxysubsection{Detailed Description}
\mbox{\hyperlink{struct_moves}{Moves}} contains all the data types and methods used to handle moves. 

\begin{DoxyAuthor}{Author}
Paul Toffoloni 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
August 19, 2021
\end{DoxyDate}
\mbox{\hyperlink{struct_move}{Move}} is a move available to a \mbox{\hyperlink{struct_tower}{Tower}}, it contains an origin cell, a destination cell, and, if the move is a conquer move, an \char`\"{}eaten\char`\"{} cell, which is the cell that gets conquered. \mbox{\hyperlink{struct_moves}{Moves}} is an array containing a list of moves.
