Checks if any moves can be performed on a determined cell.

Called when the input for the initial cell during a move has to be checked and checks if: the cell is white, meaning if it's part of the board, the cell is empty, so there is a piece in that cell, the piece in the selected cell can move somewhere. If the move is not possible, prints the reason.

Parameters
boardThe board on which the move is being made on.
xThe x coordinate of the selected cell.
yThe y coordinate of the selected cell.
turnThe turn that is being played.
Returns
0 if the move is possible, 1 if not.
