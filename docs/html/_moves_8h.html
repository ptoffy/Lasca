Detailed Description

Moves contains all the data types and methods used to handle moves.

Author
Paul Toffoloni
Date
August 19, 2021

Move is a move available to a Tower, it contains an origin Cell, a destination Cell, and, if the move is a conquer move, an "eaten" Cell, which is the cell that gets conquered. Moves is an array containing a list of moves.

Typedef Documentation

◆ Move

typedef struct Move Move

Represents a possible move for a Tower.

A user can perform a move, depending on the conquer variable, either only moves the tower in the origin cell to the destination, or also conquers the tower in the eaten cell.

◆ Moves

typedef struct Moves Moves

Array of Move.

This struct is used to find all possible moves, either for a single tower or for the whole board.

◆ conquer_tower()

void conquer_tower ( Move *  move)

Conquers a Tower from a Move.

Called when a conquest has to be performed, removes the first element of the tower to be conquered and places it on the top of the conqueror tower, then rearranges the pieces of the remaining conquered tower, if there are any left, otherwise, removes it from the cell setting its pointer to NULL.

Parameters
moveThe Move we are using to conquer the tower.
